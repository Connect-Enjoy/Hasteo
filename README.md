# Hasteo – College Bus Attendance System 🚌📱
Hasteo is a comprehensive web-based Progressive Web Application (PWA) that streamlines college bus attendance management through barcode scanning and real-time tracking. The system facilitates efficient coordination between security personnel and administrators to ensure seamless student transportation management.

---

## 🚀 Project Overview  
Hasteo revolutionizes student bus attendance management by:
- Enabling security personnel to scan student IDs for quick attendance recording (scan.html)
- Providing real-time tracking of bus arrivals and departures (bus_logs.html)
- Empowering administrators to manage student records and security personnel (student_management.html, security_management.html)
- Offering role-based authentication with persistent login (admin-login, security-login)

**Built With:**
- Frontend: HTML5, CSS3, Vanilla JavaScript (PWA-ready)
- Backend: Python Flask with PostgreSQL
- Database: PostgreSQL (Neon)
- Scanner: Instascan for barcode recognition
- UI: Modern, responsive interface with mobile-first design
- PWA Features: Service Workers, Manifest, Offline Support

---

## 🔑 Key Features

### 🛡️ Security Personnel Portal
- Scan student ID barcodes with format: COURSE/ID/YEAR
- Record bus arrival and departure timestamps in real-time
- View personal scanning history and statistics
- Offline scanning capability with automatic sync when online
- Batch processing for rapid student check-ins

### 👨‍💼 Admin Management Dashboard
- Add, edit, and manage student records with registration validation
- Create and manage security personnel accounts
- View comprehensive attendance analytics and reports
- Monitor bus operations and security performance
- Track fee status and student bus assignments

### 🎯 Student Management System
- Structured registration number format: COURSE/ID/YEAR
- Bus assignment and fee tracking
- Attendance history per student
- Active/inactive status management
- Department, year, and course tracking

### 📊 Real-time Monitoring & Analytics
- Live dashboard statistics for both admin and security roles
- Today's entry/exit counts and total scans
- Active bus tracking and status monitoring
- Historical data viewing with search functionality
- Performance metrics for security personnel
